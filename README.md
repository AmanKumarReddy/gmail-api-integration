# Gmail API Integration using Python

This project demonstrates how to connect to Gmail using OAuth 2.0 and fetch email data using Python.

## Features
- OAuth 2.0 Authentication
- Token-based access (`token.pickle`)
- Fetch latest emails (subject + sender)

## Tech Stack
- Python
- Gmail API
- OAuth 2.0
